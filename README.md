# Pause & Breathe

A Chrome extension that adds a 10-second pause before visiting distracting websites.

## Features

- Block any domain (e.g. `reddit.com`, `youtube.com`)
- Add path exceptions to allow specific pages through without a pause (e.g. block `facebook.com` but allow `facebook.com/messages`). (This was the feature missing from [Pause - Stop Mindless Browsing](https://chromewebstore.google.com/detail/pause-stop-mindless-brows/ljfdccdjpfjpfjbpdiihanpodilolofh) that made me LLM-generate this extension instead.)

## Install

1. Go to `chrome://extensions`
2. Enable _Developer mode_
3. Click _Load unpacked_ and select this directory

## Development

Run `npm install` to set up linting, formatting, and pre-commit hooks.
