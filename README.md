# Pause & Breathe

A Chrome extension that adds a 10-second mindfulness pause before visiting distracting websites.

## Features

- Block any domain (e.g. `reddit.com`, `youtube.com`)
- Add path exceptions to allow specific pages through without a pause (e.g. block `facebook.com` but allow `/messages`). (This was the feature missing from [Pause - Stop Mindless Browswing](https://chromewebstore.google.com/detail/pause-stop-mindless-brows/ljfdccdjpfjpfjbpdiihanpodilolofh) that made me LLM-generate this extension instead.)

## Install

1. Go to `chrome://extensions`
2. Enable **Developer mode** (top right)
3. Click **Load unpacked** and select this directory
